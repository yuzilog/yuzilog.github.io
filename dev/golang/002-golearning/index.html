golang快速入门(2) 数据类型

本节主要介绍golang的数据类型的分类，让给大家对数据类型有个简要的概念，俗话说的好。基础不牢，地动山摇。

二、细说类型 链接到标题

2.1、 值类型说明 链接到标题

go的变量类型包括字符串、整形、浮点、布尔，下面我们看一个简单的例子

package main

import (
	"fmt"
)

func main() {
	
	// 字符串可以用加号连接
	// golang
	fmt.Println("go" + "lang")
	
	// 数值类型操作
	// 1 + 1 = 2
	fmt.Println("1 + 1 =", 1+1)
	// 7.0 / 3.0 =  2.3333333333333335
	fmt.Println("7.0 / 3.0 = ", 7.0/3.0)
	
	// bool类型操作
	// false
	fmt.Println(true && false)
	// true
	fmt.Println(true || false)
	// false
	fmt.Println(!true)
}

2.2、 变量 链接到标题

go语言的变量名由字母、数字、下划线组成，首个字符不能为数字。

声明变量的关键字为var:

// 声明单个变量
var identifier type
// 声明多个变量
var identifier1, identifier2 type

实例说明

// 定义变量
var a = "hello"
// 定义多个变量
var b, c int = 1, 2

// 定一个不赋值的变量，
var e int
// 没有初始化情况下，声明的变量为零值，int类型类型的零值为0
fmt.Println(d) // 0

// := 的语法，是声明和初始化的简写方式
e := "yuzl" // 等价于 var e string = "yuzl"

2.3、 常量 链接到标题

go的常量类型支持字符串、布尔、数值

// 常量声明
const identifier [type] = value  // 如：const LENGTH int = 10

实例说明

// 显式类型定义 
const b string = "abc"
// 隐式类型定义编译器自动推导类型
const b = "abc"

// 常量还可以做枚举方式
const (
    Unknown = 0
    Female = 1
    Male = 2
)

本节主要是简要说明golang的常量定义，理清楚了就可以，当然里面还有很多数值类型，大家在尝试中多多练习。
